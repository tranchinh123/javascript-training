## How to run

- Clone source

```
git clone https://github.com/tranchinh123/javascript-training.git
```

- Git checkout

```
git checkout feat/typescript-practice
```

```
cd practice-one
```

- Install packages

```
npm i
```

- Run project

```
npm run build
```

```
npm start
```
